A booking portal for students to reserve lab equipment timeslots, view calendar availability, and receive automated email reminders. The project combines calendar UI widgets, RESTful endpoints, and a scheduling engine with conflict detection.

This repository contains the source code and documentation for the Remote Lab Reservation System project, developed as part of the UE23CS341A course at PES University.
